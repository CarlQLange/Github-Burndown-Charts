Github Burndown Charts

A burndown chart for your github repository. Bam.

Using Scrum? Don't want to look at a spreadsheet all day to sort out your burndown chart? I've got you covered. What you need:

  • A public github repo for your project.
  • Issues set up and in use, using milestones, on that github.
Paste the URL to your repo below, select the date you started working and the date your next milestone is to be hit. (CARL! Get this from the API, you idiot) That's pretty much it, I think. Yay MVP!
